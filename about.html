⚙️ About This Platform

Mario's Cyber Toolkit is a 100% static web application — no backend, no server, no tracking. All security operations (entropy calculation, password generation, AES encryption) run inside your browser using the Web Crypto API. It is hosted on GitHub Pages and the full source is available on GitHub. Feel free to fork, learn from, or contribute to the project.
